What This System Does

It helps researchers review uploaded molecules, score them, and surface candidates that are most useful to test next based on confidence, uncertainty, novelty, and experiment value.

How It Works

Upload data → validate columns → score or generate → rank → review

The current pipeline uses RDKit features, calibrated classification, and structured ranking logic.

What You Can Upload

CSV files with a smiles column and optional fields such as labels, molecule IDs, source, and notes. The guided upload flow helps map columns before analysis begins.

What Outputs You Receive

Ranked candidates, review-ready statuses, explanations, warnings, provenance, and structured JSON artifacts for discovery output, analysis reports, and review queues.

Who It Is For

This interface is for researchers who want support deciding which molecules deserve testing or review next, especially when screening or comparing candidate sets.

Current Limitations

Predictions are model-guided recommendations, not experimental truth. Performance is strongest for chemistry similar to the training data, and low-quality or weakly labeled uploads reduce recommendation quality.

What it does not claim

This is not a truth engine, simulator, autonomous invention platform, or replacement for chemists. It should be used to prioritize testing and review, not to replace experiments.
